# 1.0.0

February 2025

Version 1.0.0 of ClearAligner is here. Upgrade now for long-term stability and additional features.

## New Feature Highlights

* Concordance View: Navigate source by lemma.
* Notes: view and edit notes on alignment records.
* Sync: improvements to multi-user synchronization scenarios.
* Visual distinction between `content` and `function` source tokens.
* The default project is now the Berean Standard Bible.

## How To Upgrade

**Note:** _Always_ backup your project alignment data using the "export" feature before upgrading.

* Upgrading from `0.0.34`?
  * Export local projects to JSON
  * Run `1.0.0` installer
  * Wait for the local migration to complete.
* Upgrading from `0.0.33` or older?
  * Export local projects to JSON
  * Delete local ClearAligner directory
  * Install `1.0.0`
  * Re-import projects.
